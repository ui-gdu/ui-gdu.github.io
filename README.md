# GDU Website

The website for the Game Developers United club at the University of Idaho.

## Modifying the Website

Modifications that are pushed to GitHub are automatically built for the GitHub Page (website).

## Updating Event Date

To update the event date and details, modify the [`./src/JamDetails.tsx`](./src/JamDetails.tsx) file.

## Adding Jam Games

To update the past jam games, modify the [`./src/JamGames.tsx`](./src/JamGames.tsx) file.
